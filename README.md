# mp-free
An open-source hi-fi network music player. Bit-perfect FLAC playback up to 24-bit / 192 kHz, managed from your phone over WiFi. Fully FOSS — hardware, firmware, and app.   ⚠️ Status: pre-alpha. This project is in early bring-up. Expect things to be incomplete, to change, and to break. Follow along or jump in.
